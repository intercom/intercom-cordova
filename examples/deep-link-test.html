🔗 Deep Link Test Page

Test deep links for the MyShop Cordova app with Intercom integration

Instructions:
  1. Make sure the MyShop app is installed on your device
  2. Click any link below to test deep linking
  3. The app should open and navigate to the appropriate screen
  4. If the app doesn't open, check the installation and URL scheme configuration
Note: These links will only work if you have the MyShop app installed with the custom URL scheme "myshop://" configured.

📱 Basic Navigation

🏠 Open App (Home)
myshop://

myshop://profile
